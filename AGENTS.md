# Project Instructions

- After every code or project-file change, run an appropriate validation command before reporting completion. For this WPF project, default to `dotnet build SmoothScroll.sln` and confirm that it finishes without errors.
- For UI design and control usage, follow the Windows app design guidance at https://learn.microsoft.com/en-us/windows/apps/design/ and its subpages. Use https://github.com/microsoft/WinUI-Gallery as the primary WinUI control/style reference and https://github.com/CommunityToolkit/Windows as the CommunityToolkit reference. Prefer Fluent/WinUI patterns, clear hierarchy, restrained surfaces, and use `SettingsCard`/`SettingsExpander` only where they match the intended Windows settings pattern.
